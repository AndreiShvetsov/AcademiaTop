    Базой данных в данном примере является документ .htpasswd

        Форма регистрации
    Проверяется при запросе
Имя - наличие записи, наличие в базе данных
Телефон - наличие записи, соотвествие формату телефона, наличие в базе данных
Почта - наличие записи, соотвествие формату почты, наличие в базе данных
Пароль - наличие записи, наличие повторной записи, соотвествие записей между собой

    Добавление зарегестрированного пользователя
Каждый новый пользователь с новой строки
Пароль хешируется
